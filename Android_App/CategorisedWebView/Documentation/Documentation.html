Welcome to Categorized WebView

Categorised WebView - Web to Android app has ability to convert a website into an Advanced Android app in just a few minutes by adding URL into Backend and then it will be live. This WebView android app can be managed by a powerful PHP Admin Panel where you can put multiple URLs in a Single app and also you can categorised URL by there usage. Categories WebView - Web to Android app has many features like live geolocation, Downloading and uploading files, JavaScript enabled, Dedicated Desktop mode to see full size website in your Android devices a list of features are available below...

Features of Categorised WebView App

  • Easy to reskin with step by step documentation available with this code
  • No coding or programming knowledge required
  • Easy to Change colors
  • Support Multiple URLs Adding features
  • Admin Can add or Change URLs remotely without updating app source code
  • Notification feature to get updated users
  • Facebook Audience Network and Admob Ads
  • Downloading and uploading file feature available
  • Geo Location feature available
  • Easy to setup Admin Panel in any type of hosting
  • Music and Video file playing feature available
  • Dark and light mode available

  • Swipe to reload feature available
  • Admin can add website with featured option or without featured option
  • SMS, Email, Dialer, Facebook Messenger Feature available

How to use Categorised WebView App

Using Categorised WebVeiw app - Web to Android app is very easy and simple just install Admin panel in any type of web hosting and then add API URL into app source code and then you are ready to go, after building r generating app bundle or APK you can able to add website URLs into Admin panel. All the URLs will be store in the Admin panel where admin can change or update URLs remotely without changing app source code. Admin can also add own ads in this app.

Downloading and Installing Android Studio

First of all you need to download and install Android Studio. You can download it from here. After downloading you need to install it. You can follow the steps below to install it.

Windows

To install Android Studio on Windows, follow these steps:

If you downloaded an .exe file (recommended), double-click to launch it.

If you downloaded a .zip file:

Unpack the .zip. Copy the android-studio folder into your Program Files folder. Open the android-studio > bin folder. Launch studio64.exe (for 64-bit machines) or studio.exe (for 32-bit machines). Follow the Setup Wizard in Android Studio and install any recommended SDK packages. As new tools and other APIs become available, Android Studio notifies you with a pop-up. To manually check for updates, click Help > Check for Update. check the below screen shot for reference.

Mac OS

To install Android Studio on your Mac, follow these steps:

Launch the Android Studio DMG file. Drag and drop Android Studio into the Applications folder, then launch Android Studio. Choose whether to import previous Android Studio settings, then click OK. Complete the Android Studio Setup Wizard, which includes downloading the Android SDK components that are required for development.

Importing Project into Android Studio

First of all you need to extract the downloaded zip file. After extracting you will get a folder named "CategorizedWebView". Open Android Studio and click on "Open an existing Android Studio project" and select the "CategorizedWebView" folder. After selecting the folder Android Studio will start importing the project. After importing the project you will get a screen like this. Now you are ready to go. You can start editing the project. You can also check the below screenshots for reference.

Changing App Name and Package Name

To change the app name and package name you need to open the file named "strings.xml" under values directory and then you need to change the value of the string named "app_name". You can check the below screenshot for reference.

<string name="app_name">WebView App</string>

For changing the package name you need to go to project panel and then click on Settings icon. Here you need to click on Tree Appearance and then you need to uncheck Compact Middle Packages. After that you need to click on Project and then you need to click on Project Structure. And then select package name which you want to rename now you need to right click on the directory then refactor and then rename, now click on All Directories. You can check the below screenshots for reference.

Package Name > Refatcor > Rename > All Directories

After doing this you need to open App (Module) level build.gradle file and then you need to change the value of applicationId and namespace. You can check the below screenshot for reference.

applicationId "com.yourdomain.categorizedwebview"
namespace "com.yourdomain.categorizedwebview"

Change App Colors

To change the app colors you need to open the file named "colors.xml" under values directory and then you need to change the value of the color named "colorPrimary". You can check the below screenshot for reference.

<color name="colorPrimary">#3F51B5</color>

Change App Icon

To change the app icon you need to open the file named "ic_launcher.png" under mipmap directory and then you need to replace the icon with your own icon. You can also generate your own icon by right click on app in the project panel and then click on New > Image Asset. You can check the below screenshot for reference.
